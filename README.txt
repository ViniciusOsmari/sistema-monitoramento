# sistema-monitoramento
Sistema de monitoramento do uso de salas e laboratórios e controle da movimentação de equipamentos

Disponibilização dos arquivos desenvolvidos durante o Trabalho de Conclusão de Curso.

Para o funcionamento da plataforma Web, que é executada com o XAMPP, é necessário alterar os dados de acesso ao banco de dados no arquivo "codigo.bd_conexao.php".

Para o funcionamento da unidade de monitoramento no Raspberry Pi, é necessário alterar os dados de acesso ao banco de dados nos arquivos "ConexaoBD_Rpi.py" e "ConexaoBD_Servidor". Para executar o sistema de monitoramento é necessário executar o arquivos "IniciaSistema.py"
